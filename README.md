# BGDC Website

Here is how you can run the project locally:

1. Clone the repository

    ```
    git clone https://github.com/binusgdc/Website2.0.git && cd Website2.0
    ```

2. Install dependencies

    ```
    npm install
    ```

3. Start the dev server

    ```
    npm run dev
    ```
